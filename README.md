1.Класс Name
Предназначен для хранения и представления полного имени человека.

Поля (private): surname, firstName, patronymic (все типа std::string).

Конструкторы: Реализованы четыре конструктора для гибкого создания объекта:

Name() - пустое имя.

Name(const std::string& first) - только имя.

Name(const std::string& sur, const std::string& first) - фамилия и имя.

Name(const std::string& sur, const std::string& first, const std::string& pat) - полное ФИО.

Методы:

toString() const - формирует строку с ФИО, добавляя только те части, которые были заданы.

Print() const - выводит ФИО в консоль.

2. Класс Time
Предназначен для хранения времени (в пределах суток) и работы с ним.

Поле (private): totalSeconds - хранит общее количество секунд с начала суток.

Конструкторы:

Time() - 0 секунд.

Time(int seconds) - из секунд.

Time(int hours, int minutes, int seconds) - из часов, минут и секунд.

Методы:

Normalize() - приватный метод, который приводи время к диапазону [0, 86400).

getHours(), getMinutes(), getSeconds() - возвращают соответствующие компоненты времени.

toString() const - форматирует время в виде ЧЧ:ММ:СС (часы, минуты и секунды выводятся с ведущими нулями).

Print() const - выводит время в консоль.

3. Класс Employee (Сотрудник)
Представляет сотрудника, который имеет имя и может быть привязан к отделу.

Поля (private):

name (тип Name) - имя сотрудника (композиция).

department (указатель на Department) - отдел, в котором работает сотрудник (агрегация).

Конструкторы: Позволяют создать сотрудника с именем и, опционально, с привязкой к отделу. При создании с отделом, сотрудник автоматически добавляется в этот отдел.

Методы:

Геттеры для полей (getName, getDepartment).

Setdepartment(Department* dept) - для смены отдела.

toString() const - формирует подробную информацию о сотруднике: его имя, отдел и имя начальника (если он есть и если сотрудник не является начальником).

Print() const - выводит информацию о сотруднике.

4. Класс Department (Отдел)
Представляет отдел компании, в котором есть название, список сотрудников и начальник.

Поля (private):

name (std::string) - название отдела.

employees (std::vector<Employee*>) - список сотрудников отдела (агрегация).

manager (указатель на Employee) - начальник отдела.

Конструкторы: Позволяют создать отдел с именем или без.

Методы:

Setmanager(Employee* mgr) - назначает начальника. Автоматически добавляет его в список сотрудников отдела.

Addemployee(Employee* emp) - добавляет сотрудника в отдел. Проверяет, есть ли он уже в списке, и автоматически устанавливает у сотрудника указатель на этот отдел.

getEmployeeList() const - формирует строку со списком всех сотрудников отдела, помечая начальника.
